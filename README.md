# TicTacToe Mobile App
Tic Tac Toe cross-platform mobile app. There are two modes: vs computer (minimax algorithm) or vs player. 

## Software Requirements
- Xamarin 
- Visual Studio 2017
- Xamarin.Form v2.5.0
